# Discord Bot
this is a discord bot that can do many things it has a level system right now but more will be added soon

## How to use
1. clone the repo
2. run pip install -r requirements.txt
3. put your token in main.py last line
4. run main.py
5. enjoy

## How to add commands
1. go to main.py
2. go to the bottom
3. make your command
